# Ciphore Tap

## How do I install these formulae?

`brew install ciphore/tap/<formula>`

Or `brew tap ciphore/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "ciphore/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
